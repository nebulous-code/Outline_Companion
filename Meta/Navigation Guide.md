I've set up some hot keys to make navigating this vault easier. 

*`Alt` and `Option` are used interchangeably here and are the same key across Windows and Mac respectively.*
## Outline Companion

- `Alt+Shift+U` - Trigger Outline Companion

Trigger the Outline Companion script manually using the command palette `Cmd+P` or `Ctrl+P` then search for `Templater: Insert Outline Companion`

## Navigate Windows: 

Use alt/option and vim motions to navigate through multiple windows: 
- ⌥ J - Focus Window Above
- ⌥ K - Focus Window Below
- ⌥ H - Focus Window Left
- ⌥ L - Focus Window Right

Split windows with hyphen or slash
- ⌥ - - Split Window Down (horizontal)
- ⌥ \ - Split Window Right (vertical)

Close windows with X
- ⌥ x - Close Current Tab
- ⌥ + shift + x - Close Window
- Cmd + shift + T - Reopen tabs

Navigate Tabs in Windows
- Alt + D - Navigate Back (Previously opened file in tab)
- Alt + F - Navigate Forward (Formerly opened file in tab)
- Alt + B - Navigate Tab to Right
- Alt + V - Navigate Tab to Left

## Command and Open

Some ways to run commands or open files
- Cmd + P - Command Pallet
- Cmd + Shift + P - Open File 
- Alt + P - Shows File in File explorer
	- Helps you navigate the file explorer with arrow keys

## Pomodoro Navigation

Flexible Pomodoro Timer is a Plugin on this vault. 
- Cmd + O - Start a Pomodoro
- Cmd + Shift + O - Toggle Timer Pause 
	- This doesn't seem to affect the pomodoro timer
- Alt + O - Start White Noise 

## Views

These aren't used much but they're set up
- Cmd + R - Toggle Reading View 
- Cmd + Shift + R - Toggle Source View 
	- Useful if something gets formatted weird in the markdown
