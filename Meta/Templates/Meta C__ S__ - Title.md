This is an example of the template that I typically use for the metadata of my chapters.

Linked Chapter: [[C_ S_ - Title]]
## 1 Sentence Synopsis:
1 sentence about this scene
## Long Form Summary:
- What happened first?
## POV Character: 
Use # POV / CharacterFirstName
## Characters:
Use # Char / FirstName_LastName
- Description

## Setting:
Use # Location / Location_Name
- Location Description
### Weather:
- Weather Description
## Equipment, Tech or Magic:
Use # ETM / Equip_Tech_Magic_Category
- Description of the ETM
